# 🐍 [2026-01] 나만의 파이썬 소프트웨어 개발 프로젝트

## 1. 시나리오 제목
* 스마트 소비 습관 분석 및 지출 관리 시스템
<br>


## 2. 시나리오 (5~10줄)
이 프로그램은 대학생의 소비 습관을 분석하여 효율적인 지출 관리를 돕는 시스템이다.
사용자는 하루 또는 특정 기간 동안의 지출 금액과 카테고리(식비, 카페, 쇼핑, 교통 등)를 입력한다.
시스템은 입력된 데이터를 기반으로 총 지출 금액과 카테고리별 소비 비율을 계산한다.

또한 특정 기준을 초과한 지출이 발생하면 “과소비 경고”를 출력하며, 소비 패턴을 분석해 절약이 필요한 항목을 알려준다.
추가적으로 소비 습관에 따라 “절약형”, “균형형”, “과소비형”과 같은 유형을 판별한다.

최종적으로 사용자는 자신의 소비 패턴을 확인하고, 개선 방향을 제시받을 수 있다.
<br>


## 3. 예상 기능 및 메뉴 (최소 5개)
1.지출 데이터 입력 (금액 + 카테고리)
2.총 지출 금액 계산
3.카테고리별 소비 비율 출력
4.과소비 경고 시스템 (기준 초과 시)
5.소비 유형 분석 (절약형 / 과소비형 등)
6.전체 소비 내역 출력
7.프로그램 종료

## 🟩 [2차 과제: V1.0] 입출력 + 리스트 + 조건문 - 향후 작성 예정
### **✨2차 과제 내용:**
이번 과제는 '스마트 소비 습관 분석기'의 핵심 로직을 구현하는 것으로, 사용자로부터 지출 데이터를 입력받아 리스트에 저장하고, 이를 분석하여 과소비 여부와 소비 유형 등급을 판정하는 시스템을 구축하는 것입니다.

input()을 통해 카테고리별(식비, 카페, 쇼핑, 교통) 지출 금액을 입력받아 리스트에 저장하고, sum() 함수 및 산술 연산자를 활용하여 총 지출액을 산출합니다.

if-elif-else 조건문을 활용하여 총 지출 금액에 따라 '절약형(A)', '균형형(B)', '과소비형(C)'의 3단계 소비 등급 판정 로직을 구현하였습니다.

특히 and/or 논리 연산자를 활용하여, 특정 카테고리(예: 카페)의 지출이 전체의 50%를 초과할 경우 '카페 중독 주의'와 같은 특별 경고 메시지를 출력하는 지능형 진단 기능을 추가했습니다.
    
### **🤖 AI 파트너십 과정**
내용 1: 실생활 밀착형 소비 등급 산출 로직 도출

프롬프트 요약: "대학생의 하루 평균 용돈을 기준으로, 리스트와 조건문을 사용해 '현명한 소비자'인지 '과소비자'인지 판별하는 재미있는 기준을 만들어줘."

적용 내용: AI가 제안한 '총액 기준 등급'과 '특정 항목 편중 지출 체크' 로직을 결합하여, 단순히 돈을 많이 쓰는 것뿐만 아니라 특정 분야에 치우친 소비 습관까지 잡아내는 복합 조건문을 코드에 반영함.

내용 2: 데이터 구조 최적화 및 확장 로드맵 자문

프롬프트 요약: "파이썬 리스트에 4가지 카테고리 지출을 담고 싶은데, 나중에 반복문이나 딕셔너리로 발전시키기 좋은 구조가 뭐야?"

적용 내용: 초기 설계 단계에서 단순히 변수를 나열하는 대신, expenses = [식비, 카페, 쇼핑, 교통] 형태의 리스트 구조를 선택함. 이를 통해 '데이터 입력 및 리스트 저장(2차) → 반복문을 활용한 무한 입력 시스템(3차) → 딕셔너리 및 파일 저장을 통한 월간 분석(4차)'으로 이어지는 체계적인 확장 계획을 세움.
    
### **🛠️ Troubleshooting & 기술 회고:**
문제 1: 리스트 내부 요소 합산 시 발생한 계산 오류

원인: input()으로 입력받은 값들을 리스트에 넣었을 때, 숫자(int)가 아닌 문자열(str)로 저장되어 sum() 함수 사용 시 오류가 발생함.

해결: 리스트에 값을 추가하기 전 int() 함수를 사용하여 정수형으로 형변환(Type Casting)을 거친 뒤 저장함으로써 산술 연산이 가능하도록 수정함.

문제 2: 복합 조건문(and, or) 사용 시 우선순위 혼동

원인: "총액이 5만원 이상이거나, 혹은 특정 항목이 3만원 이상일 때"를 체크하는 조건문에서 괄호를 제대로 사용하지 않아 엉뚱한 결과가 출력됨.

해결: 논리 연산자의 우선순위를 학습하고, 명확한 조건 판별을 위해 (조건 A) or (조건 B)와 같이 괄호를 적절히 사용하여 로직의 정확도를 높임.
